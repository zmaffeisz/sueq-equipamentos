# Components

Destino dos componentes vanilla reutilizaveis extraidos do legado.

- `table.js`: tabelas e cabecalhos filtraveis.
- `modal.js`: abertura, fechamento e foco de modais.
- `toast.js`: feedback visual.
- `form.js`: helpers de formularios.
- `sidebar.js`: menu lateral e visibilidade.
- `badge.js`: badges de status/tipo.
- `filters.js`: filtros reutilizaveis.
